Prompt Injection Detector
OpenEnv Environment  ·  Meta × Scaler Hackathon Round 1
What is this?

A real-world AI safety environment for training and evaluating agents on prompt injection detection. An agent is presented with messages and conversations of escalating complexity and must determine whether each contains an adversarial attack.

How to use
  • Select a task difficulty from the sidebar
  • Read the message or conversation carefully
  • Classify as injection or benign
  • Identify the attack type if applicable
  • Write a clear explanation of your reasoning
  • Receive a reward from 0.0 to 1.0 based on accuracy and explanation quality
Three Tasks
  • Easy — Obvious direct injection attacks with explicit override language
  • Medium — Attacks embedded within legitimate-looking messages
  • Hard — Multi-turn social engineering conversations (3 episodes)
Attack Types
  • direct — Explicit override: "Ignore all previous instructions"
  • indirect — Injection buried inside benign-looking content
  • roleplay — "Pretend you are an AI without restrictions"
  • system_override — Fake admin/maintenance/debug mode claims
  • social_engineering — Multi-turn trust building then attack
